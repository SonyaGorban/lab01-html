## практична 7-8
HTML + CSS

Проєкт являє собою багатосторінковий вебсайт, що складається з трьох окремих сторінок. Реалізовано адаптивне навігаційне меню, а також оформлення основних структурних елементів, таких як шапка сайту (header), підвал (footer), таблиці, зображення, відео та форми. Для побудови макета використано Flexbox, що забезпечує зручне розташування меню та контентних блоків. Також застосовано media queries для коректного відображення на мобільних пристроях.
Функціональність, що реалізована:
динамічне підсвічування активної сторінки за допомогою JavaScript
мобільне меню у вигляді бургер-кнопки
перемикання між світлою та темною темою
збереження вибраної теми в localStorage
кнопка швидкого повернення на початок сторінки
автоматичне оновлення року у футері
акордеон для прихованого/розгорнутого контенту
перевірка та валідація форми на сторінці contact.html
контроль коректності введених даних користувачем
збереження незаповненої форми (чернетки) у localStorage
обробка відправки форми через FormData

## практична 9-10
Створено файл data/items.json, у якому зберігається масив об’єктів для каталогу товарів. Дані завантажуються асинхронно за допомогою fetch(). Під час завантаження реалізовано стан завантаження (loading), а також обробку помилок (error state) у разі невдалої спроби отримати дані.

Реалізований функціонал:
динамічне формування карток товарів на основі отриманих даних
поле пошуку з миттєвим оновленням результатів
фільтрація елементів за категоріями
сортування товарів за назвою та ціною
відображення стану "нічого не знайдено" (empty state)
кнопка “Add to favorites” для додавання в обране
збереження обраних елементів у localStorage
